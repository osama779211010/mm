# مكان وضع نماذج الذكاء الاصطناعي (AI Models)

يرجى وضع ملفات النماذج التي قمت بتنزيلها في هذا المجلد `assets/models/`.

## التسمية المقترحة للملفات:
لضمان عمل الكود بشكل صحيح، يفضل إعادة تسمية الملفات كالتالي:

1.  **الالتهاب الرئوي**: `pneumonia_model.tflite`
2.  **سرطان الثدي**: `breast_cancer_model.tflite`
3.  **تخطيط القلب**: `ecg_model.tflite`
4.  **كسور العظام**: `bone_fracture_model.tflite`
5.  **أورام الدماغ**: `brain_tumor_model.tflite`

> **ملاحظة:** إذا كانت الملفات بصيغة `.h5` أو `.pth`، سنحتاج لخطوة إضافية لتحويلها أو رفعها على السيرفر (Backend). حالياً، هذا المجلد مخصص للنماذج التي ستعمل داحل التطبيق مباشرة (On-Device).
